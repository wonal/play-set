How to play Set

The goal of the game is to make "sets" of 3 cards based on certain criteria. There are 81 unique cards in a deck, and the board will start out with 12 cards. Each card has four characteristics:
- color (red, yellow, purple)
- shape (circle, diamond, rectangle)
- shading (solid, hollow, patterned)
- number of shapes (one, three, six)

In order to make a set, you must select 3 cards that either ALL match in one of the four characteristics or that ALL differ in one of the four characteristics.

Once a set is made, the three cards are removed from the board and replaced with cards from the deck. The goal is to go through the deck and make as many sets as possible. If there is not a set present on the board, 3 more cards will be added.
