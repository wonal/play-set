How to play Set

There are 81 unique cards in a deck, and the board will start out with 12 cards. Each card has four characteristics:
- Color (orange, green, blue)
- Shape (circle, diamond, rectangle)
- Shading (solid, hollow, patterned)
- Number of shapes (one, three, six)

The goal of the game is to make "sets" based on certain criteria. A set is three cards where, for each of the four characteristics (color, shape, shading, number), either:
- All cards match in the characteristic
- All cards differ in the characteristic

While all cards have the same number of shapes and all cards differ in color and shading, two cards have the same shape but the other has a different shape.

Once a set is made, the three cards are removed from the board and replaced with cards from the deck. The goal is to go through the deck and make as many sets as possible. If a set is not present on the board, three more cards will be added.
